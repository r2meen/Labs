## 📖 Introduction
This repository contains solutions to various lab exercises for the **Object-Oriented Programming (OOP)** course using C++.  

The purpose of these programs is to demonstrate and understand the implementation of core OOP concepts such as:
- Classes and Objects  
- Encapsulation  
- Inheritance  
- Polymorphism  
- Abstraction  

Each program is written in a simple and structured manner to help understand the logic and application of OOP principles.

---

## 🎯 Objectives
- Understand fundamental OOP concepts  
- Apply OOP principles in C++ programs  
- Develop problem-solving skills using object-oriented design  

---

## ⚙️ Contents
The repository includes solutions related to:
- Class and Object implementation  
- Constructors and Destructors  
- Function Overloading  
- Operator Overloading  
- Inheritance (Single, Multiple, Multilevel)  
- Polymorphism (Compile-time & Runtime)
